Latihan 2

Latihan Turun Angkot

Rulesnya

1. Ketika angkot kosong, tampilkan pesan bahwa "Angkot Kosong"

2. Jika ada penumpang namanya yang sama, hapus nama penumoang dalam array dengan memberikan nama undefined

3. Jika tidak ada nama yang sama, tampilkan pesan "Bahwa tidak ada nama tersebut diangkot"



Test Case 1

Jika penumpang kosong lalu kita hapus seorang penumpang maka akan ada informasi bahwa "Angkot Kosong"



Test Case 2

Terdapat penumpang bernama ["Sandika", "Galih", "Dody"];



Test Case 3

Jika kita menyuruh orang yang tidak berada didalam angkot(Contoh : Nova ) makan tampilkan pesan "Nova tidak berada didalam angkot"
