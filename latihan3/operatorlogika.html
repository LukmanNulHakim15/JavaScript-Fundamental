Operator Logika && (AND)

Untuk operator (&&) AND dimana kondisi harus 2 bernilai True untuk menghasilkan True. Jika hanya salah satu maka nilainya False. Contoh : Var x = 8 -> (x % 2 == 0) && (x<10) maka yang dihasilkan adalah True. Karena (x % 2 ==0) && (x<10) dibaca 8 dibagi 2 namun yang dilihat sisa bagi "Apakah sisa bagi 8 dibagi 2 sama dengan 0" yang menghasilkan nilai true (AND) 8 kurang dari 10 yang memiliki nilai True

Operator Logika || (OR)

Untuk operator (||) OR dimana kondisi hanya satu yang bernilai True maka akan menghasilkan nilai True walaupun memiliki nilai False

Operator Logika ! (NOT)

untuk operator (!) NOT kondisinya terbalik. Jika pernyataan benar maka akan menghasilkan nilai False. Contoh x = 10 -> !(x < 20) maka memiliki nilai False
