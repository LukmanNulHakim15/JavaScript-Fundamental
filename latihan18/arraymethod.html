array Method

Array Method terbagi jadi beberapa

1. Length adalah method array untuk mengetahui ada berapa jumlah element didalam array

2. Join adalah method array untuk menggabungkan seluruh array dan mengubah menjadi string dan merapihkan element array

3. Push adalah menambahkan element array di akhir urutan array

4. Pop adalah kebalikan dari push yaitu menghilangkan element terakhir dari array

5. Unshift adalah menambahkan element array di urutan pertama

6. Shift adalah menghapus element pertama array

7. Splice adalah Menghapus dan menambahkan index baru sesuai posisi yang diinginkan

8. Slice adalah Mengambil index yang ada dan menjadikan array baru

9. forEach dan Map sama-sama untuk looping namun jika forEach tidak mengembalikan array sedangkan map mengembalikan array

10. Sort adalah method array yang memiliki fungsi mengurutkan arrray

11. Filter adalah method array yang memiliki fungsi mencari nilai array dan menjadikan arry

12. Find adalah method array yang sama seperti filter namun tidak dijadikan array
